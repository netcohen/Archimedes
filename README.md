# Archimedes

A local-first autonomous PC agent MVP.

## Architecture

Archimedes consists of three main components:

- **core** (C# .NET 8): Core agent logic and task execution
- **net** (Node.js TypeScript): Network layer and Firebase integration
- **android** (Kotlin): Mobile companion app for approvals and monitoring

## Project Structure

```
Archimedes/
├── core/          # C# .NET 8 backend
├── net/           # Node.js TypeScript network layer
├── android/       # Kotlin Android app
└── docs/          # Documentation
```
